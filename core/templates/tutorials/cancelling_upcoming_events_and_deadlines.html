{% extends 'tutorial_base.html' %} {% block tutorial_title %} {% block tutorial_title_tab %} Cancelling Upcoming Events & Deadlines {% endblock %} {% endblock %} {% block instructions %}

Step 1: On your homepage, find the upcoming event or dealine which you wish to cancel.

Step 2: Click "Cancel" on the bottom right of the event or deadline snippet.

Note that you can only cancel upcoming events, not past ones.

You can also click the "Cancel" button in the event section of any upcoming event or deadline page.

Step 3: Confirm the cancellation by clicking "Yes, cancel it".

Please remember that the action is final and cannot be undone.

Step 4: View all of your cancelled events by clicking "Browse" and then "Cancelled".

{% endblock %} {% block things_to_note %}
  • All RSVPs or participants of the event or deadline (respectively) which you cancel will be emailed, notifying them of the cancellation.
  • If you wish to change the time or date of an event, never cancel the existing one and create a new one. RSVPs will be notified of the cancellation and will assume that the event or deadline is no longer happening. Instead, simply edit the event.
