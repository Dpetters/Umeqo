{% extends 'tutorial_base.html' %} {% block tutorial_title %} Checking Students In {% endblock %} {% block instructions %}

Step 1: From your homepage, click on an upcoming event to which you wish to check students in.

Step 2: Click the big "ENABLE" button under "Check-In Mode".

Step 3: Click "F11" on your keyboard to make the check-in screen full-screen.

Step 4: Click on the message at the bottom right of the screen to hide it.

Step 5: See the check-in count at anytime by moving the mouse to the bottom right corner.

Step 6: Raffling away prizes? Randomly query the checkins for winners by clicking "raffle winner!".

Step 7: Exit check-in mode by clicking "exit check-in" mode.

Step 8: Follow the "Browse Attendees and RSVPs Tutorial" to browse through attendees.

{% endblock %} {% block things_to_note %}
  • It is possible, and even encouraged, to set up this check-in screen on multiple laptops at once.
  • The laptop which you use for checkins must be connected to the internet the entire time.
  • If for some reason a student cannot check-in, first check if the line below the input fields turns red when he/she clicks enter. If it does, then they have already checked-in. If it does not, check whether you are still connected to the internet.
